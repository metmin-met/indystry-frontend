# indystry.works — frontend (public mirror)

Read-only, un-branded mirror of the Astro frontend behind **https://indystry.works**
— for design reference (e.g. Claude Design). Sanitised; no deployment configs, no secrets.

- Framework: Astro + Tailwind
- Pages: `src/pages/` (DE) + `src/pages/en/` (EN)
- Views/components: `src/components/views/`
- Design tokens: `src/styles/global.css`
- Layout: `src/layouts/Base.astro`

The live site is the source of truth; this mirror is published for design tooling only.
